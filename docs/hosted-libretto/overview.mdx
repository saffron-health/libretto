> Use the managed Libretto platform to authenticate users, deploy workflows, investigate failures, and manage billing.

This section covers the hosted platform. If you want to self-host, go to the [self-hosting overview](/hosting/introduction).

<Warning>
  These pages document the Hosted Libretto account and billing flows configured by the Libretto team.
</Warning>

### Typical lifecycle

1. Ask the Libretto team for your `LIBRETTO_API_URL`.
2. Authenticate yourself and invite other users into your organization.
3. Issue API keys and deploy workflows to the hosted platform.
4. Investigate failed runs from the hosted debug emails the platform sends.
5. Manage plans and usage.
