> Set up Libretto and configure your AI provider.

### Setup

Run this in your project directory to install Libretto and configure everything:

```bash theme={null}
npm init libretto@latest
```

This walks you through:
1. Installing the `libretto` package
2. Creating the `.libretto/` directory
3. Copying agent skill files into your project
4. Configuring an AI provider for snapshot analysis
5. Downloading Chromium (if not already installed)

### AI provider

Set your snapshot analysis model:

```bash theme={null}
npx libretto ai configure <openai|anthropic|gemini|vertex>
```

| Provider    | Default model                   | Environment variable              | SDK package            |
| ----------- | ------------------------------- | --------------------------------- | ---------------------- |
| `openai`    | `openai/gpt-5.4`                | `OPENAI_API_KEY`                  | `@ai-sdk/openai`      |
| `anthropic` | `anthropic/claude-sonnet-4-6`   | `ANTHROPIC_API_KEY`               | `@ai-sdk/anthropic`   |
| `gemini`    | `google/gemini-3-flash-preview` | `GEMINI_API_KEY`                  | `@ai-sdk/google`      |
| `vertex`    | `vertex/gemini-2.5-pro`         | `GOOGLE_CLOUD_PROJECT`            | `@ai-sdk/google-vertex` |

Set the environment variable for your provider in your shell or a `.env` file at the project root, and install the matching SDK:

```bash theme={null}
npm install @ai-sdk/openai  # or whichever provider you chose
```

You can pin a specific model version:

```bash theme={null}
npx libretto ai configure openai/gpt-4o
```

To view the current configuration, run `npx libretto ai configure` with no arguments. To clear it, pass `--clear`.

### Config file reference

**Location:** `.libretto/config.json`

```json theme={null}
{
  "version": 1,
  "ai": {
    "model": "openai/gpt-5.4",
    "updatedAt": "2026-01-01T00:00:00.000Z"
  },
  "viewport": { "width": 1280, "height": 800 }
}
```

<ParamField path="version" type="number" required>
  Config schema version. Always `1`.
</ParamField>

<ParamField path="ai.model" type="string">
  A `provider/model-id` string. This is the model Libretto sends screenshot and HTML data to when you run `snapshot`.
</ParamField>

<ParamField path="viewport" type="object">
  Default browser window size (width and height in pixels) used by `open` and `run` when you don't pass `--viewport`. Built-in default is `1366x768`.
</ParamField>

<ParamField path="windowPosition" type="object">
  Optional. Sets the initial position of the headed browser window on screen. Has no effect in headless mode.
</ParamField>
