Install Libretto into the Node.js package that contains your browser automations. Run Libretto commands with `npx libretto` from that package.

Already have a Node.js package and just want to install Libretto without scaffolding? [Skip to install into an existing package](#install-into-an-existing-package).

## Create a new package

Use this if you want Libretto to scaffold a new package with directory structure, example workflows, and dependencies. This works whether you're starting fresh or adding Libretto as a new package to a monorepo.

<Steps>
  <Step title="Scaffold the package">
    ```bash theme={null}
    npm create libretto@latest
    ```

    You can also pass a package name directly:

    ```bash theme={null}
    npm create libretto@latest my-project
    ```

    This scaffolds a package directory, installs dependencies, downloads Chromium, sets up the `.libretto/` directory, and copies agent skill files.

  </Step>

  <Step title="Run a workflow">
    The scaffolded package includes an example workflow. Run it with:

    ```bash
    npx libretto run src/workflows/star-repo.ts
    ```

  </Step>
</Steps>

## Install into an existing package

If you already have a Node.js package that you plan to use for Libretto automations, you can skip the scaffolding and just install Libretto directly.

<Steps>
  <Step title="Install Libretto and Zod">
    ```bash
    npm install libretto zod
    ```
  </Step>

  <Step title="Run setup">
    ```bash
    npx libretto setup
    ```

    This downloads Chromium, creates the `.libretto/` directory, and copies agent skill files.

  </Step>

  <Step title="Create a workflow">
    Create a workflow file — for example, `src/workflows/scrape-page.ts`:

    ```ts src/workflows/scrape-page.ts
    import { workflow } from "libretto";
    import { z } from "zod";

    export default workflow(
      "scrape-page",
      {
        input: z.object({}),
        output: z.object({
          title: z.string(),
        }),
      },
      async ({ page }) => {
        await page.goto("https://example.com");
        const title = await page.title();
        console.log(`Page title: ${title}`);

        return { title };
      },
    );
    ```

    The `workflow()` function wraps your automation with browser lifecycle management, input validation, error handling, and session recording.

  </Step>

  <Step title="Run it">
    ```bash
    npx libretto run src/workflows/scrape-page.ts
    ```

  </Step>
</Steps>
